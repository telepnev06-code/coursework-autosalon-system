## 📌 Описание проекта
Данный проект является курсовой работой по дисциплине **«Технологии программирования»**. Это интеллектуальная информационная система для автоматизации бизнес-процессов автосалона, специализирующегося на продаже новых и подержанных автомобилей.

Проект реализован в парадигме **Vibe-coding** — разработка велась с применением современных ИИ-инструментов (**Gemini**) для проектирования архитектуры, оптимизации логики подбора и написания кода.

---

## ⚙️ Функциональность

### 1. Управление базой автомобилей
* **Полный учет:** Поддержка новых и подержанных отечественных и иностранных авто.
* **Технические детали:** Хранение данных о марке, годе выпуска, технических характеристиках и особенностях исполнения (комплектация).
* **Мониторинг:** Отслеживание технического состояния и запрашиваемой цены.

### 2. CRM-система (База покупателей)
* Хранение контактных координат клиентов.
* Фиксация требований: предпочтения по маркам, состоянию и техническим параметрам.
* Учет финансовых возможностей клиента.

### 3. Интеллектуальный модуль (Big Brain Matching)
* **Автоматизация подбора:** Программный поиск оптимальных вариантов авто под бюджет и запросы покупателя.
* **Логистика:** Автоматическое формирование заявок для внешних поставщиков и перегонщиков (если нужной модели нет в наличии).

## 🚀 Запуск программы

1. Склонируйте репозиторий:
   ```bash
   git clone [https://github.com/ваш-логин/auto-showroom-system.git](https://github.com/ваш-логин/auto-showroom-system.git)
   ```
2. Откройте проект в **Visual Studio**.
3. Убедитесь, что установлен компонент «Поддержка C++/CLI».
4. Скомпилируйте проект в конфигурации **Debug/Release (x86/x64)**.
5. Запустите `.exe` файл.
