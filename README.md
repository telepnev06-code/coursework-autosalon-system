## 📌 Описание проекта
Курсовая работа по дисциплине **«Технологии программирования»**. Приложение представляет собой автоматизированное рабочее место (АРМ) менеджера автосалона для учета транспортных средств и работы с клиентской базой.

**Особенность разработки:** Код спроектирован в парадигме **Vibe-coding** с использованием современных инструментов ИИ-ассистирования для оптимизации структуры и повышения читаемости кода.

---

## ⚙️ Функциональные возможности

### 1. Учет парка автомобилей
* **Единая база:** Регистрация новых и подержанных автомобилей (отечественного и иностранного производства).
* **Технические карточки:** Хранение данных о марке, годе выпуска, характеристиках, комплектации и текущем техническом состоянии.
* **Финансовый учет:** Контроль запрашиваемой цены и рыночной стоимости.

### 2. Управление клиентской базой
* **Реестр покупателей:** Ведение контактных данных и истории взаимодействий.
* **Система требований:** Фиксация пожеланий клиента к марке, техническим параметрам и состоянию авто.
* **Бюджетирование:** Учет финансовых возможностей клиента для точного подбора.

### 3. Автоматизация процессов
* **Алгоритм подбора:** Автоматическое сопоставление имеющихся в наличии автомобилей с требованиями и бюджетом покупателя.
* **Логистический модуль:** Генерация заявок для внешних поставщиков и перегонщиков в случае отсутствия нужных моделей в базе.

## 📂 Структура репозитория
* `/src` — исходные файлы приложения (.cpp, .h)
* `/data` — файлы базы данных (хранение информации об авто и клиентах)
* `/docs` — документация к проекту

## 🚀 Запуск проекта

1. Склонируйте репозиторий.
2. Откройте проект в **Visual Studio**.
3. Убедитесь в наличии рабочей нагрузки **"Разработка классических приложений на C++"** и компонента **"Поддержка C++/CLI"**.
4. Соберите решение (Build Solution) и запустите исполняемый файл.
