# Информационная система «Автосалон»

## Описание
Курсовая работа по созданию системы учета новых и подержанных автомобилей. Приложение позволяет управлять базой данных транспортных средств и клиентов, а также автоматизировать подбор авто под бюджет покупателя.

Автосалон. База новых и подержанных отечественных и иностранных автомобилей: марка, год выпуска, технические характеристики, особенности исполнения, техническое состояние, запрашиваемая цена. База покупателей: контактные координаты, требования к марке, техническим характеристикам и техническому состоянию, финансовые возможности, автоматизация подбора вариантов для покупателя, формирование заявки для поставщиков и перегонщиков.

## Функциональность
- Разработка графического интерфейса (GUI) через код.
- Программное создание таблиц и кнопок управления.
- Добавление/удаление автомобилей из базы.
- Поиск и фильтрация по марке и цене.
- Сохранение данных в текстовые файлы.

## Технологии
- **Язык:** C++
- **Интерфейс:** Windows Forms (программная реализация)
- **Среда:** Antigravity / Visual Studio

## Запуск
1. Скомпилируйте проект в режиме CLR (Common Language Runtime).
2. Запустите исполняемый файл.
