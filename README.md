📌 Описание проекта
Курсовая работа представляет собой интеллектуальную систему учета и подбора автомобилей. Проект автоматизирует работу менеджера автосалона, связывая базу доступных транспортных средств (новых и б/у) с конкретными запросами и финансовыми возможностями покупателей.

Особенность реализации: Проект разработан в парадигме Vibe-coding с использованием Gemini AI для генерации архитектурных решений и оптимизации алгоритмов на C++.

⚙️ Функциональность
В соответствии с техническим заданием, система реализует:

Управление парком авто:

Детальный учет: марка, год, тех. характеристики и особенности исполнения (комплектация).

Мониторинг технического состояния для подержанных автомобилей.

Динамическое ценообразование.

CRM-модуль (База покупателей):

Хранение контактных данных и истории обращений.

Формирование «карточки требований»: от типа двигателя до лимита по бюджету.

Интеллектуальный подбор (Matching Engine):

Автоматическое сопоставление базы авто с фильтрами покупателя.

Алгоритм ранжирования вариантов по принципу «цена-состояние».

Логистический модуль:

Автоматическое формирование заявок для поставщиков (на новые авто).

Генерация заданий для перегонщиков (на поиск конкретных моделей за рубежом).

🛠 Технологии
Язык: C++ (Managed Extensions)

Платформа: .NET / Windows Forms (CLR)

AI-Assistant: Google Gemini (проектирование логики и промпт-инжиниринг)

Хранение данных: Структурированные текстовые БД (Serialization)

🚀 Запуск и использование
Откройте решение в Visual Studio.

Убедитесь, что выбран компонент C++/CLI Support.

Скомпилируйте проект.
